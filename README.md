#  Library Management System

##  Description

A web-based Library Management System built using HTML, CSS, and JavaScript. It helps manage books, users, and issue/return operations in a simple and efficient way.

---

##  Features

* Add, update, and delete books
* Issue and return books
* Search functionality
* User-friendly interface

---

##  Tech Stack

* HTML
* CSS
* JavaScript

---

##  Usage

1. Open the project folder
2. Run `index.html` in your browser

---

##  How It Works

* Uses HTML for structure
* CSS for styling
* JavaScript for functionality and data handling
